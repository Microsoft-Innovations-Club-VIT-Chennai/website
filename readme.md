<a name="Installation" />

## Installation

**Cloning repository and installing packages**

```bash
git clone https://github.com/Microsoft-Innovations-Club-VIT-Chennai/website
npm install
cd frontend
npm install
```

**Running the project**

```bash
// at the base directory
npm run dev
```
